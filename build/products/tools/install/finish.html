The automatic installation process is complete. There are still three steps that we recommend you perform to complete the installation:

  1. Put ${installer.output.aspectjrt} on your CLASSPATH. This 29KB .jar file contains classes that are required by any program compiled with the ajc compiler.
  2. Modify your PATH to include ${installer.output.dir.bin}. This will make it easier to run ajc.
  3. Download the documentation and examples from http://aspectj.org to learn how to use the language.

These steps are described in more detail in ${installer.output.readme}.
